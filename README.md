**DkScript Pro** is the official VS Code extension for the DkScript programming language. It provides a complete development environment for building beautiful, readable, and highly efficient applications with the unique DkScript syntax.

---

## ЁЯЪА Key Features

- **ЁЯМИ Pro Syntax Highlighting**: Full color support for DkScript's unique keywords (`draft`, `solid`, `flow`, `mold`, etc.) across all VS Code themes.
- **тЪб Smart Snippets**: Rapidly generate code structures (loops, classes, functions) with simple shortcuts.
- **тЦ╢я╕П One-Click Run**: Includes a "Play" button in the editor title to execute your `.dk` files instantly.
- **ЁЯзй Language Configuration**: Supports automatic bracket matching, auto-closing pairs, and smart indentation.
- **ЁЯУБ File Icon Support**: Automatically identifies `.dk` files with the official DkScript logo.

## тМия╕П How to Use

### 1. Running Your Code
You can execute your DkScript files in three easy ways:
1. **Play Button**: Click the тЦ╢я╕П icon in the top-right corner of any `.dk` file.
2. **Context Menu**: Right-click inside the editor and select **DkScript: Run Current File**.
3. **Command Palette**: Press `Ctrl+Shift+P` and type `DkScript: Run File`.

### 2. Auto-Complete (Snippets)
Boost your productivity by using these common prefix shortcuts:

| Prefix | Output | Purpose |
| :--- | :--- | :--- |
| `mold` | `mold Class { ... }` | Create a new class |
| `flow` | `flow Name() { ... }` | Create a function |
| `when` | `when (condition) { ... }`| If statement |
| `thru` | `thru (draft i = 0; ...)`| For loop |
| `show` | `show("...");` | Print to console |
| `make` | `make ClassName();` | Create an instance |

## ЁЯУЪ Keyword Documentation

DkScript Ultra syntax provides 100% JavaScript parity with a clean, expressive English vocabulary:

- **Variables**: `draft` (let), `solid` (const)
- **Functions**: `flow` (top-level), `step` (internal method), `emit` (return)
- **Logic**: `when`, `other`, `match`, `site`
- **Objects**: `mold`, `onto`, `start`, `self`, `make`
- **Asynchronous**: `quick` (async), `pause` (await)
